# Twibbon Creator

Aplikasi web untuk membuat twibbon dengan fitur lengkap.

## Fitur

- 📤 **Upload Foto** - Upload foto dari perangkat
- 🖼️ **Frame Gradient** - Pilih berbagai frame gradient keren
- ✏️ **Teks Overlay** - Tambahkan teks dengan berbagai style
- 🎨 **Filter Foto** - 8 filter efek foto
- ⭐ **Stiker & Emoji** - 16 stiker emoji yang bisa diposisikan
- ✂️ **Crop Tool** - Potong foto dengan presisi
- 📐 **Resize** - Ubah ukuran foto untuk media sosial
- 🔄 **Geser & Zoom** - Atur posisi dan zoom foto dengan drag
- 🧹 **Background Removal** - Hapus background foto otomatis
- 💾 **Simpan Template** - Simpan dan muat pengaturan template

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- react-easy-crop
- @imgly/background-removal

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Deploy

Aplikasi ini bisa di-deploy ke Vercel, Netlify, atau platform hosting lainnya.

```bash
# Deploy ke Vercel
npx vercel
```
